# simple-space



Игра Simple-Space
Жанр: Песочница, Симулятор.
Платформы: Linux, Windows

Описание:
В игре вам даётся запустить ракету в любой уголок солнечной системы!

Особенности:
- 10 видов космических кораблей
- Вся Солнечная система
- Реалистичная физика
